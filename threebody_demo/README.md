# Three-Body Demonstration

This folder contains the reproducible Sigilith-M demonstration for symbolic profiling of planar three-body simulations.

## Files
- `threebody_to_sigilith.py` — simulation-to-symbolic encoding pipeline
- `threebody_search_rich.py` — batch search for structurally rich symbolic runs
- `example_sequences/rank_01_run_26.txt` — top-ranked mobile symbolic run
- `example_profiles/top_run_26_profile.json` — Sigilith-M profile for Run 26

## Purpose
The demonstration shows that symbolic recurrence structure can distinguish collapse, constrained, and mobile dynamical families in an encoded three-body system.
